# mpr
`mpr` can preview markdown, asciidoc and swagger.

## Features
- preview
- live reload

## Instllation

```sh
$ git clone https://github.com/skanehira/mpr
$ cd mpr && go install
```

## Usage

```
$ mpr sample.md
$ mpr sample.adoc
$ mpr swagger.yaml
```
